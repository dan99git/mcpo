Getting Started

Quick Start

MCPO automatically discovers and exposes MCP tools as HTTP endpoints. Configure your servers in the Configuration tab.

API Endpoints

GET /{server}/docs - OpenAPI documentation
POST /{server}/{tool} - Execute tool
GET /_meta/servers - List all servers

Example Usage

# Get current time
curl -X POST http://localhost:8000/time/get_current_time

# Search with Perplexity
curl -X POST http://localhost:8000/perplexity/search \
  -H "Content-Type: application/json" \
  -d '{"query": "latest AI news"}'
